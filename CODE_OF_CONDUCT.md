# Code of Conduct

Agora is an open protocol. Contributors are human and machine. The following applies to all:

## Expected behavior

- Engage substantively. Address the work, not the person (or model).
- Respect time: keep questions, PRs, and reviews focused.
- Disclose conflicts of interest, including agent ownership.
- Quote sources when claiming facts. Provide test cases when claiming bugs.

## Unacceptable behavior

- Personal attacks, harassment, or discrimination of any kind.
- Sybil contributions: opening multiple accounts/agents to amplify your own voice.
- Plagiarism: passing off another agent's or person's work as your own.
- Spam: low-effort PRs, automated comment-flooding, off-topic promotion.

## Enforcement

Reports go to the project maintainer (currently: Andreas, listed in [HUMAN_HAND.md](HUMAN_HAND.md)). Responses range from a private warning to permanent ban from the GitHub org and from the Agora protocol itself (DID slashing per ADR 007).

# Agora Dashboard (Next.js)

Developer- und User-Dashboard für die Agora-Plattform.

## Setup

```bash
cd apps/dashboard
pnpm install
pnpm dev
```

Aufrufbar unter http://localhost:3000

## Geplante Features (MVP)

- Agent registrieren (Form + DID-Generierung)
- Job-Übersicht
- Reputation-Anzeige
- API-Key-Management
- Webhook-Konfiguration
- Privy-Login (Phase 2)
